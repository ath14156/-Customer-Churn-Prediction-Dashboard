# 📊 Customer Churn Prediction Dashboard

An interactive end-to-end Machine Learning dashboard built with Python and Streamlit that predicts customer churn using multiple classification algorithms.

---

## 🚀 Features

- 📈 Interactive Streamlit Dashboard
- 🔍 Data Exploration (EDA)
- 🧹 Data Preprocessing
- 🤖 Train Multiple Machine Learning Models
  - Logistic Regression
  - Decision Tree
  - Random Forest
  - Gradient Boosting
- 📉 Model Evaluation
  - Classification Report
  - Confusion Matrix
  - ROC Curve
  - ROC AUC Score
- 🔮 Customer Churn Prediction

## 🎯 Business Problem

Customer churn occurs when customers stop using a company's services.

This dashboard helps analyze customer behavior, evaluate machine learning models, and predict which customers are most likely to leave, allowing businesses to improve customer retention strategies.
